## 📋 Sobre

Portfólio pessoal desenvolvido para apresentar trajetória profissional, projetos de BI e competências técnicas. O site é 100% estático (HTML + CSS + JS vanilla), sem dependências de frameworks, com foco em performance e compatibilidade.

**Destaques:**
- Dashboards Power BI embeddados (iframe interativo)
- Toggle de idioma PT/EN
- Seção de skills com barras animadas (hard + soft skills)
- Timeline profissional consolidada (estilo LinkedIn)
- Animação de contagem nos indicadores
- CV ATS-friendly disponível para download (.docx)
- Meta tags SEO e Open Graph

## 🚀 Como Usar

Não há build, não há dependências. Basta abrir o arquivo:

```bash
# Clone o repositório
git clone https://github.com/wfcs/portfolio.git
cd portfolio

# Abra no navegador
start main.html        # Windows
open main.html         # macOS
xdg-open main.html     # Linux
```

Ou sirva com qualquer servidor local:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```
